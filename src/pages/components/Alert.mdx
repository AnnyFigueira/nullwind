# Alert

These are system-generated notifications with a high degree of criticality. They should be used when the system is unstable or there is a risk of loss of data or access to resources.

## Usage

```jsx
import { Alert } from "nullwind";

const MyComponent = () => {
  return (
    <Alert color="info" ondismiss={() => {}}>
      This is a info alert
    </Alert>
  );
};
```
