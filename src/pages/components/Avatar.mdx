# Avatar

Avatars are used to represent a person or organization. They can be used in a variety of places, including navigation, lists, tables, cards, and more.

## Usage

```jsx
import { Avatar } from "nullwind";

const MyComponent = () => {
  return (
    <Avatar
      src="/avatar.avif"
      name="John Doe"
      description="Joined in April 1976" // optional
      placeholder="avatar-placeholder.png" // optional
    />
  );
};
```
