# Quick Start

## Installation

```bash
yarn add -D tailwindcss @tailwindcss/forms @tailwindcss/typography
yarn add nullwind
```

Nullwind is a modern UI component library for Tailwind CSS, you just need to install Tailwind CSS and configure it.

## Usage

```tsx
import Nullstack from "nullstack";
import { ThemeProvider, Button } from "nullwind";
import type { Theme } from "nullwind";

// Custom theme (Optional)
const customTheme = {
  button: {
    base: "rounded-md shadow-sm",
  },
} as Theme;

export default class Application extends Nullstack {
  render() {
    return (
      <ThemeProvider theme={customTheme} persistent>
        <body>
          <Button>Hello world!</Button>
        </body>
      </ThemeProvider>
    );
  }
}
```

## Configuration

You need to add this to your `tailwind.config.js` file.

```js
// tailwind.config.js
const defaultTheme = require("tailwindcss/defaultTheme");
const colors = require("tailwindcss/colors");

module.exports = {
  content: ["node_modules/nullwind/src/theme.ts", "src/**/*.{jsx,tsx}"],
  theme: {
    extend: {
      // Set font family
      fontFamily: {
        sans: ["Inter", ...defaultTheme.fontFamily.sans],
      },
      // Set theme colors (Required config!)
      colors: {
        primary: colors.pink,
        secondary: colors.gray,
        info: colors.blue,
        success: colors.green,
        warning: colors.orange,
        danger: colors.red,
      },
    },
  },
  // Add plugins
  plugins: [require("@tailwindcss/typography"), require("@tailwindcss/forms")],
};
```

## Theme

We add two colors to the theme, primary and secondary, you can set your colors. This is a required configuration, please configure it according to your own needs.

1. First you net import the colors from tailwindcss/colors.

```js
const colors = require("tailwindcss/colors");
```

2. Then you can set your colors, more colors can be found in Tailwind CSS Customizing Colors.

```js
// Set theme colors (Required config!)
colors: {
  primary: colors.blue,
  secondary: colors.slate,
},
```

3. You can also use Tailwind CSS Color Shades Generator to produce and configure your colors.

```js
// Set theme colors (Required config!)
colors: {
  primary: {
    50: "#e6f6ff",
    100: "#b3ebff",
    ...
  },
  secondary: {
    50: "#f4f5f7",
    100: "#e4e7eb",
    ...
  },
},
```

## Font

We use Inter as the default font, You can easily use Google Font CDN to import Inter fonts.

```html
<head>
  <link rel="preconnect" href="https://fonts.googleapis.com" />
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
  <link
    href="https://fonts.googleapis.com/css2?family=Inter:wght@100..900&display=swap"
    rel="stylesheet"
  />
</head>
```

Then you can add the Inter to the tailwind.config.js file.

```js
// Set font family
fontFamily: {
  sans: ["Inter", ...defaultTheme.fontFamily.sans],
},
```

To use different fonts, simply replace the font “inter” with your preferred font.

## Plugins

We use @tailwindcss/typography and @tailwindcss/forms plugins by default, you can add them to the tailwind.config.js file. It is not required, you can configure it as needed.

```js
// Add plugins
plugins: [require("@tailwindcss/typography"), require("@tailwindcss/forms")],
```
