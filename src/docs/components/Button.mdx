# Button

A button is an interface element that makes it possible to perform an action. It must clearly and
objectively represent the effect of the action when clicked. Do not use buttons to navigate between
pages, for that use Links.

## Usage

```jsx
import Button from "nullwind/Button";

const MyComponent = () => {
  return (
    <Button
      href="https://nullstack.app" // set custom `href` to use as a link
      color="primary" // primary, secondary, info, success, warning, danger
      size="md" // xs, sm, md, lg, xl
      outline={false}
      fullSized={false}
      rounded={true}
      disabled={false}
      onclick={() => {}}
    >
      Button
    </Button>
  );
};
```
