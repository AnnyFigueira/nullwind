# Tab

Use these responsive tabs components to create a secondary navigational hierarchy for your website or toggle content inside a container

## Usage

```jsx
import { Tab } from "nullwind";

const MyComponent = () => {
  return (
    <Tab onchange={(idx) => alert(`Choosen Tab: ${idx}`)}>
      <Tab.Item title="Profile" icon={IconUser} active>
        Tab content 1
      </Tab.Item>
      <Tab.Item title="Notifications" icon={IconBell}>
        Tab content 2
      </Tab.Item>
      <Tab.Item title="Settings" icon={IconSettings}>
        Tab content 3
      </Tab.Item>
    </Tab>
  );
};
```
