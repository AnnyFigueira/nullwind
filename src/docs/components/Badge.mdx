# Badge

Badges are used to highlight important information, such as the number of unread messages.

## Usage

```jsx
import Badge from "nullwind/Badge";

const MyComponent = () => {
  return (
    <Badge id="my-badge" class="my-class" color="info" size="lg">
      This is an badge
    </Badge>
  );
};
```
